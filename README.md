# Task 3: SQL for Data Analysis

## Objective
Perform SQL queries to analyze relational data using MySQL.

## Tools Used
 MySQL Server
 MySQL Workbench

## Dataset
The project uses the following tables:

 pelanggan (Customer Details)
 produk (Product Details)
 orders (Order Details)
 detil_order (Order Item Details)

## SQL Concepts Covered

 SELECT
 WHERE
 ORDER BY
 GROUP BY
 INNER JOIN
 LEFT JOIN
 RIGHT JOIN
 Aggregate Functions (SUM, AVG)
 Subqueries
 Views
 Indexes

## Queries Performed

1. Displayed records using SELECT.
2. Filtered data using WHERE.
3. Sorted records using ORDER BY.
4. Grouped records using GROUP BY.
5. Joined tables using INNER JOIN, LEFT JOIN, and RIGHT JOIN.
6. Calculated total and average order values using SUM and AVG.
7. Used a subquery to retrieve orders above the average total.
8. Created a VIEW for customer order analysis.
9. Created an INDEX to improve query performance.

Successfully analyzed the dataset using SQL by applying filtering, sorting, grouping, joins, aggregate functions, subqueries, views, and indexes. The project demonstrates practical SQL skills for relational database analysis.
